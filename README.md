# FETCH

Get the best deal

FETCH is a smart price comparison web app that compares food delivery and shopping prices across Sri Lankan platforms.

---

## Features

- Compare PickMe Food vs Uber Eats
- Compare Daraz, Kapruka & Ikman
- AI recommendation system
- Shows the cheapest option instantly
- Calculates savings automatically
- Mobile responsive UI

---

## Tech Stack

- Next.js
- Tailwind CSS
- n8n
- OpenAI GPT-4o
- Vercel
